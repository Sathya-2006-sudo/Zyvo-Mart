# ZyvoMart
      ZyvoMart is a modern full-stack eCommerce web application built using Next.js, MongoDB, and Tailwind CSS.
      It allows users to browse products, view product details, and purchase items through a clean and responsive interface.

# Features
        Product listing
        Product details page
        User authentication
        Shopping cart
        Order management
        Admin dashboard
        Fully responsive design
  
# Tech Stack
    1. Frontend
          Next.js
          React
          Tailwind CSS
    2. Backend
          Node.js
          MongoDB
          Mongoose
    Authentication
          JWT / NextAuth (if used)

# Installation & Setup
      1️. Clone the Repository
      git clone https://github.com/Sathya-2006-sudo/Zyvo-Mart.git
      2️. Navigate to Project Folder
      cd ZyvoMart
      3️. Install Dependencies
      npm install
      4. Run the Development Server
      npm run seed
      npm run dev
      5. Open in Browser
      http://localhost:3000
